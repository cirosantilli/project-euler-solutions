# Project Euler 16 Solution - Power Digit Sum

<https://projecteuler.net/problem=16>:

* [16.py](16.py)
* [16.lean](16.lean)

We need the sum of the decimal digits of \(2^{1000}\).

## Key idea
Python integers have arbitrary precision, so we can compute \(2^{1000}\) exactly using `2 ** 1000`, convert it to a string, and sum its characters as digits.

## Steps
1. Compute `n = 2 ** 1000`.
2. Convert to decimal string `str(n)`.
3. Sum `int(ch)` for each character `ch`.

A small assert validates the example: the digit sum of \(2^{15}\) is 26.

The program prints the final result.
