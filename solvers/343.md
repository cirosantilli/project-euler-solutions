# Project Euler 343 Solution - Fractional Sequences

<https://projecteuler.net/problem=343>:

* [343.py](343.py)

For a reduced state `a/b`, the sum `s = a + b` starts at `n + 1`.  After one
unreduced step, the reduction factor is

\[
\gcd(a+1,b-1)=\gcd(a+1,s).
\]

Starting a phase from `1/(s-1)`, no reduction occurs until the numerator reaches
the smallest prime factor of `s`; that reduction divides `s` by that prime and
returns the numerator to `1`.  Repeating this strips prime factors from smallest
to largest.  When only the largest prime factor remains, the chain finishes with
numerator `LPF(n+1)-1`, so

\[
f(n)=\operatorname{LPF}(n+1)-1.
\]

For the required cubes,

\[
k^3+1=(k+1)(k^2-k+1),
\]

so the summand is

\[
\max(\operatorname{LPF}(k+1),\operatorname{LPF}(k^2-k+1))-1.
\]

The first largest-prime-factor table is a normal sieve for all `k+1`.

For `Q_k = k^2-k+1`, a prime `p` divides `Q_k` when

\[
(2k-1)^2 \equiv -3 \pmod p.
\]

The prime `3` gives the single class `k = 2 mod 3`; for primes above `3`, roots
exist exactly when `p = 1 mod 3`.  The solver finds the square root of `-3` with
Tonelli-Shanks, visits the two residue classes for `k`, strips all powers of
`p` from the stored residual `Q_k`, and records `p` as the current largest prime
factor.  Any residual left after all primes up to the limit have been processed
is itself the remaining large prime factor.
