# Project Euler 880 Solution - Nested Radicals

<https://projecteuler.net/problem=880>:

* [880.py](880.py)

This solver computes

\[
H(10^{15}) \bmod (1031^3+2).
\]

## Primitive families

The nested-radical classification leaves two primitive branches.  Let
\(a,b\in\mathbb{Z}_{>0}\), \(\gcd(a,b)=1\), and exclude the degenerate line
\(a=2b\).

For odd \(b\),

\[
X=b(b+4a)^3,\qquad Y=4a(a-2b)^3.
\]

For even \(b\),

\[
X=2b\left(\frac b2+2a\right)^3,\qquad Y=a(a-2b)^3.
\]

Every admissible pair is one of these primitive seeds multiplied by a common
square \(t^2\).

## Cube-free filter

The problem excludes the cases where the two cube-root parts collapse to the
same cube-free kernel.  The solver precomputes

\[
\operatorname{cf}(n)=\prod_p p^{v_p(n)\bmod 3}
\]

with a smallest-prime-factor sieve.  A primitive seed is kept only when

\[
\operatorname{cf}(b)\ne\operatorname{cf}(4a)
\]

in the odd branch, or

\[
\operatorname{cf}(2b)\ne\operatorname{cf}(a)
\]

in the even branch.

## Bounds and summation

The positive coordinate gives the search limits

\[
b\le\left\lfloor(4N)^{1/4}\right\rfloor,
\]

\[
a\le\frac{\left\lfloor(N/b)^{1/3}\right\rfloor-b}{4}
\]

for odd \(b\), and

\[
a\le\frac{\left\lfloor(N/(2b))^{1/3}\right\rfloor-b/2}{2}
\]

for even \(b\).

For each surviving primitive seed, all square multiples are summed at once.  If

\[
T=\left\lfloor\sqrt{\frac{N}{\max(|X|,|Y|)}}\right\rfloor,
\]

then the seed contributes

\[
(|X|+|Y|)\sum_{t=1}^{T}t^2
=(|X|+|Y|)\frac{T(T+1)(2T+1)}6.
\]

The code asserts the check value \(H(1000)=2535\).
