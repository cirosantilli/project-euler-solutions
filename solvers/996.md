# Tennis Leaderboard Overtakes

This solution counts the possible overtake-count vectors without simulating match sequences.

## Main ideas

1. **Ignore non-overtake matches.**
   A non-overtake match leaves both the ranking and all counts unchanged. Therefore only the overtakes matter, and the number of overtakes must be even when the ranking returns to the identity. If `T = k // 2`, the task is to count valid vectors with total count at most `2T`.

2. **Characterize valid vectors by positive blocks.**
   Split an overtake-count vector into maximal contiguous blocks of positive entries, separated by zeros. A block with entries `x_1, ..., x_l` is valid exactly when:

   - its sum is even, say `2t`, and
   - every entry is at most `t`.

   A one-entry positive block is therefore impossible. Multiple blocks are independent because a zero gap means no player ever needs to cross that gap.

3. **Count one block with inclusion-exclusion.**
   For a block of length `l` and cost `t`, the number of positive integer vectors with sum `2t` and all entries at most `t` is

   ```text
   A(l, t) = C(2t - 1, l - 1) - l * C(t - 1, l - 1),
   ```

   where the second binomial term is treated as zero if `t < l`. At most one entry can exceed `t`, so only one subtraction term is needed.

4. **Use generating functions.**
   For each block length `l`, the series `sum_t A(l,t) q^t` has denominator `(1-q)^l`. The program stores only its numerator. A dynamic program scans positions and appends either:

   - a zero, or
   - a whole positive block starting after a zero or at the beginning.

   For a prefix of length `i`, its generating function is stored as a numerator over `(1-q)^i`, so appending a zero multiplies the numerator by `(1-q)`, while appending a block multiplies by that block's numerator.

5. **Convert exact-cost counts to cumulative counts.**
   At the end,

   ```text
   H_n(q) = N_n(q) / (1-q)^n
   ```

   counts exact costs. Summing all costs up to `T` is the coefficient of `q^T` in

   ```text
   N_n(q) / (1-q)^(n+1).
   ```

   This is evaluated with binomial coefficients. The final target is computed modulo `1234567891`.

## Running

```bash
python3 main.py
```

The program runs the two sample assertions from the problem statement, then prints the requested modular result.
