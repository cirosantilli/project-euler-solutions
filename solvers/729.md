# Project Euler 729 Solution - Range of Periodic Sequence

<https://projecteuler.net/problem=729>:

* [729.py](729.py)

The map is `f(x) = x - 1/x`.  Solving `y = f(x)` gives the two inverse
branches

\[
g_\pm(y)=\frac{y\pm\sqrt{y^2+4}}2.
\]

A period can therefore be described by choosing one of these two inverse
branches at each step and finding the fixed point of the composed inverse map.

## Primitive Branch Words

Rotating a branch word only changes the chosen starting point on the same
cycle.  If the word is a repetition of a shorter word, the orbit has a smaller
true period.  The solver therefore generates only aperiodic binary necklaces,
using the Fredricksen-Kessler-Maiorana recursion.

For length `n` it also checks the generated count against

\[
\frac1n\sum_{d\mid n}\mu(d)2^{n/d},
\]

which is the number of primitive binary necklaces.

## Fixed Point and Range

For one necklace, the composed inverse map is a contraction, so its fixed point
is unique.  The code starts Newton iteration at `0` for

\[
\Phi(x)=G(x)-x,
\]

where `G` is the branch composition.  While evaluating `G`, it also multiplies
the branch derivatives by the chain rule, giving the Newton denominator
`G'(x) - 1`.

Once the fixed point is known, the orbit is unfolded by applying the same
inverse branches.  The maximum minus the minimum is the range for that necklace.
The problem counts every cyclic starting point separately, so a length-`n`
necklace contributes `n * range`.

The final floating-point sum is accumulated in chunks with `math.fsum`; the
answer is printed rounded to four decimal places.
