# Project Euler 451 Solution - Modular Inverses

<https://projecteuler.net/problem=451>:

* [451.py](451.py)

We need the largest `m < n - 1` such that `m` is its own inverse modulo `n`.
This is equivalent to

\[
m^2 \equiv 1 \pmod n.
\]

The excluded value `n - 1` is the always-present root `-1`.

## Local Roots

For an odd prime power `p^e`, the only roots of `x^2 = 1` are

\[
x \equiv \pm 1 \pmod {p^e}.
\]

Powers of `2` are the special case:

* modulo `2`, there is only one root
* modulo `4`, the roots are `+1` and `-1`
* modulo `2^e` for `e >= 3`, the roots are `+-1` and `2^(e-1) +- 1`

## CRT Projectors

Factor

\[
n=\prod q_j
\]

into prime powers. For one factor `q`, set `M = n/q` and choose

\[
E \equiv M^{-1}M \pmod n,
\]

where the inverse is taken modulo `q`. This value is `1` modulo `q` and `0`
modulo every other prime-power factor, so it changes only the local residue for
that factor.

The code starts from the global root `-1`, represented by `n - 1`. For each
prime-power factor, it adds the lifted offsets that move the local root from
`-1` to the other local roots. The resulting list is the full set of roots of
`x^2 = 1 (mod n)`.

The answer `I(n)` is the largest generated root below `n - 1`.

## Checks

The implementation verifies the examples `I(7)=1`, `I(15)=11`, and `I(100)=51`.
