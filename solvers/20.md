# Factorial Digit Sum (100!)

## Approach
- Compute `n!` using integer multiplication in a loop. Python's integers have arbitrary precision, so `100!` is handled exactly.
- Convert the factorial value to a string and sum its digits.

## Correctness notes
- The statement provides `10! = 3628800` and digit sum `27`; we assert `factorial_digit_sum(10) == 27`.
- Then we compute and print the digit sum of `100!`.

## Complexity
- Time: `O(n * M)` where `M` is the cost of big-integer multiplication (very small for `n=100`).
- Space: `O(D)` where `D` is the number of digits of `n!` (also small for `n=100`).
