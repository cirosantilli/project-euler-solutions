# Project Euler 837 Solution - Amidakuji

<https://projecteuler.net/problem=837>:

* [837.py](837.py)

The two rung types are adjacent transpositions in \(S_3\):

\[
A=(12),\qquad B=(23).
\]

We count words with exactly \(m\) copies of \(A\) and \(n\) copies of \(B\) whose
product is the identity, modulo \(1234567891\).

## Pair Decomposition

If \(m+n\) is odd, the product is an odd permutation and cannot be the identity.
Otherwise write \(m+n=2t\) and split the word into \(t\) adjacent pairs.

The pair types are

\[
AA=e,\qquad BB=e,\qquad AB=r,\qquad BA=r^2,
\]

where \(r=(123)\). If exactly \(k\) pairs are mixed, then

\[
u=\frac{m-k}{2},\qquad v=\frac{n-k}{2}
\]

are the forced counts of \(AA\) and \(BB\) pairs. Thus \(k\) is admissible when

\[
0\le k\le\min(m,n),\qquad k\equiv m\pmod 2.
\]

For fixed \(k\), the number of pair layouts is

\[
T_k=\frac{t!}{u!\,v!\,k!}.
\]

Among the \(k\) mixed pairs, the number of orientations whose product is the
identity in the order-3 subgroup is

\[
R_k=\frac{2^k+2(-1)^k}{3}.
\]

Therefore

\[
F(m,n)=\sum_k T_k R_k \pmod {1234567891}.
\]

## Fast Evaluation

The first admissible \(k\) is \(k_0=m\bmod 2\). Its layout factor is

\[
T_0=\binom{t}{m/2}
\]

when \(k_0=0\), and

\[
T_1=t\binom{t-1}{(m-1)/2}
\]

when \(k_0=1\).

Consecutive admissible values differ by 2, and

\[
\frac{T_{k+2}}{T_k}
=\frac{(m-k)(n-k)}{4(k+1)(k+2)}.
\]

The solver walks through \(k,k+2,\dots\), updates this ratio modulo the prime
modulus, and updates \(2^k\) by multiplying by 4 each step. Modular divisions are
handled with block batch inversion, so the loop uses one modular exponentiation
per block rather than one per denominator.

The code checks the two statement values \(F(3,3)=2\) and
`F(123,321) = 172633303`.
