# Project Euler 97 Solution - Large Non-Mersenne Prime

<https://projecteuler.net/problem=97>:

* [97.py](97.py)

We need the last 10 digits of:

\[
28433 \times 2^{7830457} + 1
\]

The last 10 digits are the value modulo \(10^{10}\). So we compute:

\[
(28433 \cdot 2^{7830457} + 1) \bmod 10^{10}
\]

Key insight: use modular exponentiation to compute \(2^{7830457} \bmod 10^{10}\) efficiently in \(O(\log n)\) time.
