# Project Euler 754 Solution - Product of Gauss Factorials

<https://projecteuler.net/problem=754>:

* [754.py](754.py)

Let

\[
GF(n)=\prod_{\substack{1\le k<n\\(k,n)=1}} k .
\]

The target is \(\prod_{n\le 10^8} GF(n)\) modulo \(10^9+7\).

## Key ideas

Classify the product by divisors instead of by coprime residues. Mobius inversion gives

\[
GF(n)=\prod_{d\mid n}\left(\left({n\over d}-1\right)!d^{n/d-1}\right)^{\mu(d)}.
\]

Multiplying this over all `n <= N` and writing `n = d*m` gives

\[
\prod_{d=1}^{N}
\left(d^{q(q-1)/2}\,SF(q-1)\right)^{\mu(d)},
\quad q=\left\lfloor{N\over d}\right\rfloor,
\]

where \(SF(k)=1!2!\cdots k!\).

The quotient `q = N // d` is constant over intervals. For each interval `[l, r]`,
the code only needs:

- product of `d` with `mu(d) = 1`
- product of `d` with `mu(d) = -1`
- sum of `mu(d)` over the interval

The Mobius function is computed with a linear sieve. As each `mu(d)` becomes known,
the solver immediately folds `d` into the current quotient interval's two products
and Mobius sum. That avoids a second full scan of `1..N` just to build the interval
aggregates. Superfactorials are generated once, but stored only at the `q - 1`
values that the quotient intervals will request.

## Checks

The code verifies:

- `GF(10) = 189`
- \(\prod_{n=1}^{10} GF(n)=23044331520000\)
- the same product modulo \(10^9+7\) is `331358692`
