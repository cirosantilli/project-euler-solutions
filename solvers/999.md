# Sequence solution summary

## Main idea

The original recurrence has an alternating coefficient, which makes direct modular iteration awkward and far too slow for an index near `10**18`.

Introduce a rescaled sequence

```text
W_n = sign(n) * 2^floor(n^2 / 4) * a_n
```

where `sign(n)` repeats with period four:

```text
+1, +1, -1, -1
```

After substitution, the alternating recurrence becomes the constant-coefficient elliptic divisibility sequence recurrence

```text
W_{n+2} * W_{n-2} = 4 * W_{n+1} * W_{n-1} + 4 * W_n^2
```

with seeds

```text
W_0 = 0
W_1 = 1
W_2 = 2
W_3 = -4
W_4 = -32
```

## Fast doubling

Elliptic divisibility sequences have doubling identities analogous to Fibonacci fast doubling:

```text
W_{2m-1} = W_{m+1} * W_{m-1}^3 - W_{m-2} * W_m^3

W_{2m} = (W_m / W_2) *
         (W_{m+2} * W_{m-1}^2 - W_{m-2} * W_{m+1}^2)
```

Since `W_2 = 2` and the requested modulus is odd, division is always multiplication by the fixed modular inverse of `2`. No division by a potentially zero sequence term is needed.

The implementation recursively keeps the eight consecutive values

```text
W_{n-3}, ..., W_{n+4}
```

so each halving step can rebuild the next block with constant work.

## Recovering the requested sequence

Once `W_n` is known, recover the original value modulo the requested modulus with

```text
a_n = sign(n) * W_n * 2^(-floor(n^2 / 4))
```

## Complexity

The index is halved at every recursive step:

```text
Time:  O(log n)
Space: O(log n)
```

The program uses only built-in Python operations and includes assertions for the values supplied in the problem statement.
