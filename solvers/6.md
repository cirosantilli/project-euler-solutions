# Project Euler 6 Solution - Sum Square Difference

<https://projecteuler.net/problem=6>:

* [6.py](6.py)
* [6.lean](6.lean)

We need:
- **Sum of squares**: \(1^2 + 2^2 + \dots + n^2 = \frac{n(n+1)(2n+1)}{6}\)
- **Square of sum**: \((1 + 2 + \dots + n)^2\), where \(1 + 2 + \dots + n = \frac{n(n+1)}{2}\)

The required value is:
\[
\left(\frac{n(n+1)}{2}\right)^2 - \frac{n(n+1)(2n+1)}{6}
\]

This computes in constant time using integer arithmetic (no loops needed). For \(n=100\), the result is `25164150`.
