# Project Euler 56 Solution - Powerful Digit Sum

<https://projecteuler.net/problem=56>:

* [56.py](56.py)
* [56.lean](56.lean)

We need the maximum sum of digits of numbers of the form `a^b` where `a, b < 100`.

## Approach
- Iterate over all pairs `(a, b)` with `1 <= a < 100` and `1 <= b < 100` (only 99×99 = 9801 cases).
- Compute `a^b` using Python’s arbitrary-precision integers.
- Convert the result to a string and sum its digits.
- Track the maximum digit sum found.

This is fast enough because the search space is tiny and Python big integers handle `a^b` efficiently.

## Result
The maximum digital sum is **972**.
