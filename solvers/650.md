# Project Euler 650 Solution - Divisors of Binomial Product

<https://projecteuler.net/problem=650>:

* [650.py](650.py)

For

\[
B_n=\prod_{k=0}^{n}\binom{n}{k},
\]

we need

\[
S(N)=\sum_{n=1}^{N}\sigma(B_n)\pmod {10^9+7}
\]

at `N = 20000`.

## Factorial Form

Multiplying

\[
\binom{n}{k}=\frac{n!}{k!(n-k)!}
\]

over all `k` gives

\[
B_n=\frac{(n!)^{n+1}}{\left(\prod_{j=0}^{n} j!\right)^2}.
\]

For a prime `p`, write `A_n = v_p(n!)` and
`T_n = A_0 + A_1 + ... + A_n`. Then the exponent of `p` in `B_n` is

\[
e_{n,p}=(n+1)A_n-2T_n.
\]

Comparing this with the same formula for `n-1` gives the update

\[
e_{n,p}-e_{n-1,p}=(n-1)v_p(n)-A_{n-1}.
\]

Every prime receives the common `-A_{n-1}` shift, while only primes dividing
`n` receive the positive `(n-1)v_p(n)` correction.

## Modular State

The divisor sum is

\[
\sigma(B_n)=\prod_p \frac{p^{e_{n,p}+1}-1}{p-1}.
\]

The code stores `p^(e+1)` modulo `10^9+7` for each prime, plus the reusable
factor `p^(-v_p(n!))` that applies the common negative shift on the next step.
After updating the primes in `n`, it multiplies the local sigma factors for all
active primes and adds the result to the prefix sum.

An SPF table supplies fast factorization for each `n`.

## Checks

The solver checks the statement examples for `n = 5`, the exact value of
`S(10)`, and the known modular checkpoint `S(100) = 332792866`.
