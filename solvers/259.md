# Project Euler 259 Solution - Reachable Numbers

<https://projecteuler.net/problem=259>:

* [259.py](259.py)

The digits `1` through `9` must be used in order. Between adjacent digits we may
concatenate or use a binary operation, and parentheses may be placed arbitrarily.

## Key ideas

For any substring of the digit string, store every exact rational value reachable
from that substring. The whole substring interpreted as one decimal number is one
candidate; every split also combines a value from the left substring with a value
from the right substring using `+`, `-`, `*`, and `/`.

This gives an interval dynamic program:

\[
R(l,r)=\{\text{integer formed by digits }l..r-1\}\cup
\bigcup_{l<s<r} R(l,s)\operatorname{ op }R(s,r).
\]

The same substrings appear many times, so the interval results are memoized.
Fractions are stored as reduced `(numerator, denominator)` pairs with positive
denominators, which makes set deduplication exact and avoids floating-point
rounding errors.

At the end, only pairs with denominator `1` and positive numerator contribute to
the answer, and a final set removes duplicates across all expressions.

## Check

The code checks that `42` is reachable, matching the example expression from the
problem statement.
