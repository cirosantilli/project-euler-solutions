# Project Euler 155 Solution - Counting Capacitor Circuits

<https://projecteuler.net/problem=155>:

* [155.py](155.py)

Normalize the base capacitance to `1`.  Every equivalent capacitance is then a
positive rational number.

Let `E_n` be the set of values obtainable with exactly `n` capacitors.  A circuit
with `n` capacitors splits at its top connection into two smaller circuits with
sizes `a` and `n-a`, so the dynamic program combines all pairs from earlier
layers.

For two values `x` and `y`, parallel composition is `x + y`, while series
composition is

\[
\frac{xy}{x+y}=\frac1{1/x+1/y}.
\]

Because replacing every series node by a parallel node and vice versa maps a
unit-capacitor circuit value `v` to `1/v`, each `E_n` is closed under
reciprocal.  The solver therefore only builds the parallel sum `x+y` and inserts
both that value and its reciprocal; this covers the series case as well.

Values are stored as reduced `(numerator, denominator)` pairs, so equal
capacitances deduplicate exactly.  The answer is the size of the union of
`E_1` through `E_18`, since the problem allows at most 18 capacitors.
