# Project Euler 415 Solution - Titanic Sets

<https://projecteuler.net/problem=415>:

* [415.py](415.py)

The solver counts the complement. By Sylvester-Gallai, a subset of the grid is
not titanic exactly when it is empty, a singleton, or all selected points are
collinear.

## Key ideas

For a grid line with `k` lattice points, the non-titanic subsets on that line have
size at least 3. Rewriting the binomial contribution by line length gives a sum
over `l` of:

\[
F_l(N)(2^l-(l+1)),
\]

where `F_l(N)` is the number of grid lines with at least `l + 1` points.

Primitive direction vectors reduce `F_l(N)` to cumulative totient sums:

\[
\Phi(M)=\sum_{d\le M}\varphi(d),\quad
\Psi_1(M)=\sum_{d\le M}d\varphi(d),\quad
\Psi_2(M)=\sum_{d\le M}d^2\varphi(d).
\]

These summatory functions are precomputed up to a cutoff and then evaluated for
large arguments with quotient-block recursion from the identity
\(\sum_{d\mid n}\varphi(d)=n\).

On an interval where both `N // l` and `N // (l + 1)` are constant, `F_l(N)` is a
quadratic polynomial in `l`. The solver sums each whole interval using closed
forms for:

- \(\sum 2^l\), \(\sum l2^l\), \(\sum l^2 2^l\)
- \(\sum l\), \(\sum l^2\), \(\sum l^3\)

Finally it subtracts the collinear complement, the empty set, and all singletons
from \(2^{(N+1)^2}\) modulo \(10^8\).
