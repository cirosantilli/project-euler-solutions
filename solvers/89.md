# Project Euler 89 Solution - Roman Numerals

<https://projecteuler.net/problem=89>:

* [89.py](89.py)

To measure how many characters can be saved, each given Roman numeral is:

1. **Parsed to an integer**  
   Scan left to right; if a symbol is followed by a larger-valued symbol, treat it as subtractive (e.g. `IV = 5 - 1`), otherwise add it.

2. **Re-encoded in minimal Roman form**  
   Use a greedy decomposition with the standard minimal tokens:
   `M, CM, D, CD, C, XC, L, XL, X, IX, V, IV, I`.

3. **Count saved characters**  
   Add `len(original) - len(minimal)` across all lines.
