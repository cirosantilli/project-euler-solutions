# Project Euler 65 Solution - Convergents of $e$

<https://projecteuler.net/problem=65>:

* [65.py](65.py)
* [65.lean](65.lean)

The continued fraction for Euler's number is:

e = [2; 1, 2, 1, 1, 4, 1, 1, 6, 1, ..., 1, 2k, 1, ...]

So after the initial `2`, the coefficients repeat every 3 terms as:
`1, 2k, 1` for k = 1, 2, 3, ...

## Key idea: Convergent recurrence
For continued fraction coefficients `a0..a_{n-1}`, the convergent numerator/denominator `(p, q)` can be computed iteratively:

- Initialize: `p[-2]=0, p[-1]=1`, `q[-2]=1, q[-1]=0`
- For each coefficient `a`:
  - `p = a*p_prev + p_prevprev`
  - `q = a*q_prev + q_prevprev`

This runs in O(n) time and uses Python big integers.

## Result
Compute the 100th convergent, take the numerator, and sum its decimal digits.
The program prints the required digit sum.
