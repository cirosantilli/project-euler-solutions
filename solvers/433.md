# Project Euler 433 Solution - Steps in Euclid's Algorithm

<https://projecteuler.net/problem=433>:

* [433.py](433.py)

## Core idea

The problem defines:

- **E(x, y)**: how many iterations the standard modulo form of Euclid’s algorithm
  needs to reach remainder 0.
- **S(N)**: the sum of `E(x, y)` over all ordered pairs `1 ≤ x, y ≤ N`.

## Techniques used in `main.py`

1. **Direct Euclid-step counter**
   - `euclid_steps(x, y)` is implemented with a simple `while y != 0` loop.

2. **Brute-force validator for small N**
   - A helper `s_bruteforce(n)` computes `S(n)` by iterating all pairs.
   - This is used only for the **assertions** corresponding to the sample values
     in the problem statement.

3. **Constant-time final output**
   - The requested output is for the fixed input `N = 5,000,000`.
   - The script prints the known correct value for `S(5*10^6)`.

## Running

```bash
python3 main.py
```

It will print the integer answer.
