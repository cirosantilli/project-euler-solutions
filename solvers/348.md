# Project Euler 348 Solution - Sum of a Square and a Cube

<https://projecteuler.net/problem=348>:

* [348.py](348.py)

We need the five smallest palindromes that have exactly four representations

\[
n = a^2 + b^3,\qquad a,b > 1.
\]

## Main idea

The solver searches palindromes directly instead of generating all square-plus-cube
sums. A decimal palindrome is determined by its left half, so the candidates are
produced in increasing order by mirroring each prefix:

* even length: mirror the full prefix
* odd length: mirror the prefix except for its final center digit

Because lengths and prefixes are scanned in increasing order, the first five hits
are the five smallest valid palindromes.

## Counting one candidate

For a palindrome `n`, each possible cube `b^3` leaves a single possible square:

\[
a^2 = n - b^3.
\]

So the code keeps a growing list of cubes with `b >= 2`, extends it only when the
current palindrome is large enough, and tests each remainder with `math.isqrt`.
The cube loop stops once `b^3 + 4 > n`, because the smallest allowed square is
`2^2`.

If exactly four cube remainders are perfect squares, the palindrome is accepted.
