## Number Letter Counts (1 to 1000)

We need the total number of letters used when writing out the numbers 1..1000 in words (British usage), ignoring spaces and hyphens.

### Key ideas
- Use fixed word lists for:
  - `1..9` (ones)
  - `10..19` (teens)
  - `20,30,...,90` (tens)
- Compute the letter count directly via `len(word)` and summation.
- For `100..999`:
  - Always include `"<one..nine>hundred"`.
  - If there is a non-zero remainder, add `"and"` (British rule), then add the letter count for the remainder (1..99).
- Handle `1000` as `"one thousand"`.

### Complexity
Constant-time per number; overall `O(1000)` operations, easily within limits.
