# Project Euler 216 Solution - The Primality of $2n^2 - 1$

<https://projecteuler.net/problem=216>:

* [216.py](216.py)

Instead of testing every value `2n² - 1` for primality, the solver sieves the
indices `n`.

## Key ideas

If an odd prime `p` divides `2n² - 1`, then

\[
n^2 \equiv 2^{-1} \pmod p.
\]

This congruence has solutions only for primes `p == 1` or `7 (mod 8)`. For each
such prime up to `sqrt(2N² - 1)`, the code computes the two roots modulo `p` and
marks the corresponding arithmetic progressions of `n`.

There is one exception: if `2n² - 1` equals the sieving prime `p` itself, that
index must not be marked composite. This happens exactly when `(p + 1) / 2` is a
perfect square.

After all progressions are marked, every unmarked `n >= 2` has no prime divisor
up to the square root of `2n² - 1`, so its value is prime.

## Check

The code verifies `solve(10000) = 2202`.
