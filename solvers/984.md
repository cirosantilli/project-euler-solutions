# Project Euler 984 Solution - Knights and Horses

<https://projecteuler.net/problem=984>:

* [984.py](984.py)

## Main ideas

The important reduction is that the counting sequence is eventually a
quasi-polynomial with period 2.  Its transition polynomial factors with a
ninth-order root at `1` and a second-order root at `-1`, so after the short
initial prefix each parity class is governed by a fixed polynomial in `n`.

The requested index is `10^18`, which is even.  For even `n > 3` the required
branch is the degree-8 polynomial

```text
31/40320 n^8 + 31/3360 n^7 + 67/1440 n^6 + 41/320 n^5
+ 313/1440 n^4 - 5699/240 n^3 + 16049/420 n^2
+ 29413/140 n - 419.
```

Since the modulus `1_000_000_007` is prime and none of the denominators are
multiples of it, every rational coefficient can be evaluated by multiplying by
the denominator's modular inverse.

The solver keeps the coefficients in a compact table, builds `n^0..n^8` modulo
`1_000_000_007`, and accumulates the eight terms plus the constant.  Two
published checks are retained:

- `f(100) = 8658918531876`, using exact rational arithmetic.
- `f(10000) == 377956308 (mod 1_000_000_007)`.

## Complexity

The final computation uses a fixed number of modular multiplications and
inverses, so runtime and memory use are both `O(1)`.
