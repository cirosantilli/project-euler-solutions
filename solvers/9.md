# Project Euler 9 Solution - Special Pythagorean Triplet

<https://projecteuler.net/problem=9>:

* [9.py](9.py)
* [9.lean](9.lean)

We want natural numbers `a < b < c` such that:

- `a^2 + b^2 = c^2` (Pythagorean triplet)
- `a + b + c = 1000`

## Key Insight (Euclid's Formula)

Every Pythagorean triplet can be written as:

- `a = k (m^2 - n^2)`
- `b = k (2mn)`
- `c = k (m^2 + n^2)`

for integers `m > n >= 1` and scaling factor `k >= 1`.

Summing gives:

`a + b + c = k( (m^2-n^2) + 2mn + (m^2+n^2) ) = 2k m (m+n)`

So we need:

`2k m (m+n) = 1000`

We iterate over small `m` and `n`, compute `s = 2*m*(m+n)`, and if `1000 % s == 0` then `k = 1000 // s` produces a valid candidate triplet. We verify it and return `abc`.

## Complexity

`m` is small because `2*m*(m+1) <= total`, so the search is very fast (milliseconds).
